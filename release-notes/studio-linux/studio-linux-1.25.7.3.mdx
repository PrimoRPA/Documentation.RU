# Primo RPA Studio Linux 1.25.7.3

Раздел содержит описание изменений для версии приложения Primo RPA Studio Linux 1.25.7.3 (патч), выпущенной в июле 2025 года.

## Новые функции и улучшения

1. Реализована возможность Primo Robot работать с переменной окружения PRIMOSALT для использования паролей, зашифрованных с помощью соли.
